# Innova Mentor

**Última actualización:** Junio de 2026

## Objetivo del documento

Este documento registra los principales hitos alcanzados durante el desarrollo de **Innova Mentor**.

Su propósito es documentar la evolución del proyecto, registrar las decisiones más importantes y facilitar el seguimiento del crecimiento de la aplicación a lo largo del tiempo.

A diferencia del historial de Git, este documento resume únicamente los cambios más significativos del proyecto.

---

# Hito 1 — Nacimiento del proyecto

## Objetivo

Crear la base técnica sobre la cual se desarrollará Innova Mentor.

## Principales avances

* Creación del proyecto utilizando React y Vite.
* Configuración inicial del repositorio en GitHub.
* Organización de la estructura principal del proyecto.
* Configuración inicial de Git.
* Primer despliegue utilizando Vercel.

## Resultado

Se obtiene una primera versión funcional de la aplicación, lista para comenzar el desarrollo.

---

# Hito 2 — Construcción de la identidad

## Objetivo

Definir la identidad visual y conceptual del proyecto.

## Principales avances

* Definición del nombre oficial **Innova Mentor**.
* Creación del eslogan institucional:

> **"Aprender vuelve a ser posible."**

* Creación del personaje **Nova** como mentor virtual.
* Diseño del banner principal.
* Diseño del logotipo horizontal.
* Diseño del icono oficial para la PWA.
* Integración de Nova en la pantalla principal.
* Definición de la paleta de colores institucional.

## Resultado

El proyecto adquiere una identidad visual consistente y fácilmente reconocible.

---

# Hito 3 — Organización del proyecto

## Objetivo

Preparar el proyecto para un crecimiento ordenado.

## Principales avances

* Reorganización de la estructura de carpetas.
* Creación de la carpeta **docs**.
* Creación de la documentación técnica.
* Creación de las guías para usuarios.
* Actualización del README.
* Organización del branding y recursos gráficos.

## Resultado

El proyecto cuenta con una estructura preparada para el trabajo colaborativo y la incorporación de nuevas funcionalidades.

# Evolución del proyecto

Durante el desarrollo de Innova Mentor se mantendrá la siguiente filosofía:

* Construcción incremental.
* Documentación continua.
* Código limpio.
* Componentes reutilizables.
* Mejora continua.
* Trabajo colaborativo.

Cada nuevo hito deberá reflejar una mejora significativa del proyecto y quedar documentado tanto en Git como en este documento.

---

# Relación con Git

Los commits representan cambios individuales en el código.

Este documento registra únicamente los hitos más importantes del proyecto.

Ambos mecanismos son complementarios y permiten comprender tanto la evolución técnica como la evolución funcional de Innova Mentor.

# Conclusión

La historia de Innova Mentor no se mide únicamente por la cantidad de líneas de código escritas, sino por la evolución constante del proyecto.

Cada hito representa un paso más hacia el objetivo de construir una plataforma que ayude a las personas a recuperar el hábito del aprendizaje mediante tecnología, acompañamiento y motivación.
