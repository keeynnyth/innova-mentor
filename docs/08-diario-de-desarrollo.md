# Innova Mentor

**Última actualización:** Junio de 2026

## Objetivo del documento

Este documento registra el desarrollo diario de Innova Mentor.

Su finalidad es documentar las actividades realizadas en cada sesión de trabajo, las decisiones adoptadas, los avances obtenidos y los próximos pasos planificados.

A diferencia del Historial de Versiones, este diario refleja el proceso de construcción del proyecto y no únicamente los resultados finales.

---

# ¿Cómo se utilizará este documento?

Cada sesión de trabajo incorporará una nueva entrada siguiendo una estructura uniforme.

Cada registro incluirá:

* Fecha.
* Objetivo de la sesión.
* Actividades realizadas.
* Dificultades encontradas.
* Soluciones implementadas.
* Aprendizajes obtenidos.
* Próximos pasos.

De esta forma será posible reconstruir la evolución completa del proyecto.

---

# Sesión 1

## Fecha

Junio de 2026

## Objetivo

Crear la base del proyecto y preparar el entorno de desarrollo.

## Actividades realizadas

* Creación del proyecto utilizando React y Vite.
* Configuración inicial de Git.
* Creación del repositorio en GitHub.
* Primer despliegue utilizando Vercel.
* Configuración como Aplicación Web Progresiva (PWA).

## Aprendizajes

Se definió la estructura inicial del proyecto y se verificó el funcionamiento del flujo de desarrollo utilizando Git y GitHub.

## Próximos pasos

Comenzar el diseño de la identidad visual.

---

# Sesión 2

## Objetivo

Construir la identidad visual del proyecto.

## Actividades realizadas

* Definición del eslogan institucional.
* Creación del personaje Nova.
* Diseño del banner principal.
* Diseño del logotipo horizontal.
* Creación del icono oficial para la PWA.
* Integración del branding en la aplicación.

## Aprendizajes

La identidad visual evolucionó progresivamente hasta lograr una imagen coherente con los objetivos del proyecto.

## Próximos pasos

Mejorar la pantalla principal de la aplicación.

# Sesión 3

* Creación de la pantalla de bienvenida.

# Buenas prácticas

Durante el desarrollo del proyecto se recomienda:

* Registrar la sesión el mismo día.
* Documentar únicamente información relevante.
* Describir tanto los avances como las dificultades.
* Registrar las decisiones importantes.
* Mantener una redacción clara y objetiva.
