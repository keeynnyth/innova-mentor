
# Experiencia del Usuario

## Objetivo

Definir el recorrido completo que realiza el usuario dentro de Innova Mentor desde que abre la aplicación hasta que comienza su primer desafío.

---

# Etapas de la experiencia

1. Bienvenida
2. Registro o Inicio de sesión
3. Presentación de Nova
4. Onboarding personalizado
5. Preparación del recorrido
6. Inicio del desafío diario
7. Seguimiento del progreso
8. Celebración de logros

---

# Filosofía

La aplicación no busca únicamente enseñar contenidos.

Busca ayudar al usuario a recuperar el hábito de aprender mediante pequeños avances diarios acompañados por Nova.

---

# Rol de Nova

Nova acompaña durante todo el recorrido.

No juzga.

No presiona.

No reemplaza a un docente.

Actúa como una guía que adapta la experiencia a cada usuario.

---

# Personalización

Toda la experiencia se adapta según:

- Objetivos.
- Dificultades.
- Tiempo disponible.
- Intereses.

Esta información se obtiene durante el onboarding y permite generar un recorrido personalizado.

---

# Principios

- Cada pequeño avance cuenta.
- El progreso es más importante que la velocidad.
- La experiencia debe sentirse humana.
- La aplicación acompaña, no obliga.
